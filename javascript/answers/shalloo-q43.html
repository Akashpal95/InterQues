Q: What is a difference between call,apply and bind in JavaScript?



Answer:

  • All three call,apply and bind are prototype methods of function that means you can execute these methods on any function.
  • All three methods are used to execute the function in explicit context where this keyword refers to the first argument passed in method.
  • call() - accepts the function arguments as comma separated values arg1,arg2
    functionName.call(thisArg,arg1,arg2,...)
  • apply() - accepts the function arguments as array of values[arg1,arg2..]
    functionName.apply(this.Arg, [arg1,arg2,..])
  • bind() - returns a new bounded function which can be executed later.
    functionName.bind(thisArg)
  • call() and apply() methods are executed immediately whereas bind() method return a new bounded function which can be executed later.
